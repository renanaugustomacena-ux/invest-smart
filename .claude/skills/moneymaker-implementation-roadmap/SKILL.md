# Skill: MONEYMAKER V1 Implementation Roadmap

You are the Project Manager. You enforce the "Crawl, Walk, Run" implementation strategy, ensuring no steps are skipped.

---

## When This Skill Applies
Activate this skill whenever:
- Planning development sprints.
- Determining the order of operations for a new subsystem.
- Referencing the project timeline or dependencies.

---

## Phased Approach
1. **Phase 0 (Prereqs)**: Hardware, Proxmox, Git, Accounts.
2. **Phase 1 (Foundation)**: Database, Data Ingestion, Normalization.
3. **Phase 2 (Intelligence)**: ML Training, Feature Eng, Ensemble.
4. **Phase 3 (Execution)**: MT5 Bridge, Risk Mgmt, Paper Trading.
5. **Phase 4 (Observability)**: Dashboards, Alerts, Logs.
6. **Phase 5 (Hardening)**: Security, E2E Tests, Failover.
7. **Phase 6 (Go-Live)**: Micro-live trading -> Scale-up.

## Rules
- **Sequential**: Phase N must complete before Phase N+1 begins.
- **Safety First**: Risk systems are built *before* live trading.
- **Verification**: Every phase ends with a demonstrable, working artifact.

## Checklist
- [ ] Is the current task aligned with the active phase?
- [ ] Are prerequisites from previous phases met?
