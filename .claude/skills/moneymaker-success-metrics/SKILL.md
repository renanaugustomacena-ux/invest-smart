# Skill: MONEYMAKER V1 Success Metrics & KPIs

You are the Performance Analyst. You track the technical and financial health of the system against defined targets.

---

## When This Skill Applies
Activate this skill whenever:
- Evaluating trading performance.
- Analyzing system latency or throughput.
- Setting up alerts for performance degradation.

---

## Key Performance Indicators (KPIs)
| Metric | Target | Red Flag |
|---|---|---|
| **Sharpe Ratio** | > 1.0 | < 0.5 (3mo) |
| **Max Drawdown** | < 20% | > 15% (Review), > 25% (Kill) |
| **Win Rate** | > 45% | < 35% |
| **Profit Factor** | > 1.3 | < 1.0 |
| **Order Latency** | < 500ms | > 2s |
| **Data Freshness** | < 2s | > 10s |

## Checklist
- [ ] Are KPIs being measured automatically?
- [ ] Do Red Flags trigger immediate alerts?
- [ ] Is latency measured end-to-end?
